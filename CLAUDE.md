# CLAUDE.md

生信外包 Agent Harness 项目。开始工作前先读 `AGENTS.md`。

## 默认行为

- 常规任务以执行为目标，不只写计划
- 高风险或方法分歧大的任务，先给 3-5 条方法概要：工具/库、macOS 风险、验证方式
- 一旦用户确认方法，必须在同一会话内推进到实际执行，不要停在规划阶段

## 质量保障

- 每完成一次分析或修复，都要运行对应的验证，再报告完成
- 报告生成后必须做数值交叉校验
- 打包前必须做 AI 痕迹扫描

## 报告生成

- 所有数值必须程序化从源数据读取，禁止手写
- 编辑 Word 时优先段落匹配，不用行号定位
- 检查嵌图是否正常、图注是否匹配实际图表、数字是否与源数据一致

## 环境

- 默认环境是 macOS bash 3.x
- 禁止使用 `declare -A` 和其他 bash 4+ 特性

## Git

- 提交前先检查 `.gitignore`
- 不要在被忽略的文件上浪费提交尝试
